# Agent Instructions

## Bilingual Planning Docs

For generated design, spec, planning, or explanatory documents, create and maintain both English and Chinese versions.

- Use the English filename as the base document.
- Add a Chinese sibling with `-cn` before the extension.
- When one language version changes, update the corresponding version in the same task.
- Keep scope, decisions, requirements, and exclusions aligned between both versions.

Example:

- `docs/superpowers/specs/2026-06-04-happen-mvp-review-value-design.md`
- `docs/superpowers/specs/2026-06-04-happen-mvp-review-value-design-cn.md`
